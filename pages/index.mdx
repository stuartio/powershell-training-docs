## What is PowerShell?

[PowerShell](https://github.com/powershell/powershell) is an open-source interactive shell, originally written for Windows only but made cross platform in 2016. It offers many features over and above the default shells provided with operating systems, and can be used as both a shell and advanced scripting language.
