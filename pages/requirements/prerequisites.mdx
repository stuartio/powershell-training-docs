# Prerequisites

In order to get the most from this course, you need to install a few small applications on your own system. You _could_ do this all in a Docker container if you so choose, but leaving you with the tools to continue working with PowerShell is one of the goals of the course. The requirements differ slightly between MacOS and Windows, but there is no reason you can't work with PowerShell in any supported OS.

## MacOS

### Homebrew

Homebrew is a package manager for MacOS which is extremely useful in managing installed binaries on your Mac. The initial setup can take some time, and if there are issues they can be tricky to troublshoot, so make sure to run this well in advance of the class.

_Check homebrew is installed already_
```shell copy
where brew
```

_Install homebrew_
```shell copy
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```
